# Арт достижений (ачивок)

Формат: **WebP**. Папка: `assets/ach/`.
Если файла нет — вместо арта показывается эмодзи-заглушка на полосатом фоне,
вёрстка не ломается. Имя файла = id ачивки.

Рекомендуемый размер: **480×300 (16:10)**, тёмный фон под общий стиль UI
(золотисто-каменная палитра, как у `--gd` ≈ #c9a24b). Стиль — лаконичная
символика стоицизма, без текста на картинке.

Ачивки видны только во вкладке **«Обзор»** кабинета фракции.
